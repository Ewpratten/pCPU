# pCPU Programmer's Reference

## Data Types

| Type | Bin         | Assembler Prefix |
|------|-------------|------------------|
| I8   | `1000 0000` | `#`              |
| I16  | `0100 0000` | `##`             |
| M8   | `0010 0000` | `%`              |
| M16  | `1010 0000` | `%%`             |
| REG  | `1100 0000` | `$`              |
